# Proyecto de Análisis de Ventas

## Integrante
Valentina Fernández

## Escenario elegido
Escenario B – Análisis de Ventas de una Pequeña Empresa

## Descripción
Este proyecto realiza un análisis básico de ventas utilizando Python y Pandas.

Se trabajó con datos simulados de productos, cantidades y precios para calcular:

- Ventas totales
- Productos vendidos
- Análisis simple de datos
